# OpenCV Car detection from video stream

## How to run the code
```bash
git clone git@github.com:thegoofy-dev/OpenCV_car_detection.git
cd opencv-car-detection-video
python main.py
```

## install packages

```bash
pip install -r requirements.txt
```
